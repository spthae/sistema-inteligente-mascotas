# Sistema Inteligente para la Gestión de Mascotas

Aplicación desarrollada en Python que utiliza Machine Learning para analizar el estado emocional y de salud de las mascotas.

## Tecnologías utilizadas

- Python
- TensorFlow / Keras
- Pandas
- Scikit-learn
- Git y GitHub

## Funcionalidades

- Detección de emociones mediante una red neuronal LSTM.
- Detección de síntomas y generación de alertas de salud.
- Registro y gestión de información de mascotas.
- Estimación de la edad equivalente en años humanos según la especie.
